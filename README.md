## 📌 Featured projects

### ♟️ ChessTutor AI — AI-powered chess performance analyzer
> FastAPI · Python · Stockfish · Google Gemini · MongoDB Atlas · Docker

Uploads up to 20 PGN games at once, runs Stockfish engine analysis on every move (centipawn loss, best alternatives, blunder classification), and generates a personalized coaching report via Gemini grounded in real engine data — not generic advice.

**Key engineering decisions:** async FastAPI with Stockfish engine pool for concurrent analysis; Gemini receives actual engine output (UCI moves + cp_loss) to prevent hallucinations; MongoDB Atlas for game and user persistence.

### 🧠 Neuroscience Virtual Tutor — RAG-based AI tutoring system
> Python · FastAPI · RAG · Nomic Embeddings · MongoDB Atlas Vector Search · Gemini

University research project (Iniciação Científica) in collaboration with a master's student. Built a virtual tutor for Applied Neuroscience that answers questions and generates quizzes grounded strictly in curated academic PDFs — using RAG to eliminate LLM hallucinations in a specialized domain.

**Key engineering:** custom semantic chunking algorithm (sentence-boundary aware) for high-precision retrieval; Nomic `embed-text-v1.5` embeddings (768 dimensions) stored in MongoDB Atlas Vector Search; top-8 retrieval window for cross-source context.
